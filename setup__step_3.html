This is the screen of the final step for setup. The main purpose is for users to join in or create study groups in communities. First,  A introduction layer will appear when loading the page and contains information guiding users on the functionalities: join in communities, join in study groups and create study groups. Users can click button “ok” and close the layer. 


The main page are divided in three sections.

1) The search bar allow users to search groups within all and every communities users joined in  when users select corresponding part in navigation bar.

2) The navigation bar contains 3 buttons. join, all and community users join. (in this case it is MST).  The “join”  button enables users to go to rechoose community page and join in other community when click the join button. The “all “ and “MST” button enables users to view algorithm-generated group recommendations within corresponding communities.

3) The content section display algorithm-generated group recommendations. It is divided into 3 categories. When the “all” in navigation bar is selected, it will display 3 categories : picked for you, learn with friends and recently created.  When the “MST” in navigation bar is selected . it will display 3 categories trending in MST, Picked for you and learn with friends. 

      “picked for you “ contains groups that are recommended according to the content of the posts viewed, replied to and liked by users in the forum and the topics of the study groups in which they have participated.

        “learn with friends” contains groups that users’ friend join in.

         “recently created” contains groups that are recently created.

         “Trending in (the community  selected in nav bar, in this case it is MST) contains groups that are most viewed and with the most private messages for application received by the group owner at the time users browsing.


Users click groups to go to group info page. They can also click button “more” to see more  algotithm-generated groups. If users don’t feel like joining in existing groups , they can click the red “create groups” button and go to create group page and create groups.
